# Lycontainers
## Penjelasan
Lycontainers adalah module dari lyra dengan isi utama sebagai alat penyimpanan / data struktur untuk menyimpan data. terdapat dua kategori
dalam Lycontainers itu sendiri, yaitu:
1. Data Structure Containers    :   data struktur untuk menyimpan data - data seperti dynamic list, linked list, dan lainnya
2. Allocators                   :   data struktur untuk alokasi memori seperti linear allocators, zone allocators, dan lainnya

## Data Structure Containers
Data structure Containers memiliki tujuan untuk menyimpan data dan mengaturnya, adapun data struktur yang disediakan terdiri dari
1. Dynamic list
2. Linked list
3. Ring buffers
4.  Hashmap

## Dynamic Storage Allocations
DSA adalah perantara untuk menyimpan dan memanage memory untuk digunakan oleh operasi lainnya, adapun opsi yang disediakan adalah
1. Linear Allocators
2. Zone Allocators
3. Pool Allocators
4. TLSF
