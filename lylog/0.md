# Tujuan
jadi saya ingin membuat system macro yang dapat diubah oleh pengguna dengan define mereka sendiri, dimana konsepnya seperti ini:

1. Orang luar melakukan #define pada macro yang telah di tentukan yaitu macro A dengan dua variable yaitu log level dan log colors
2. apabila A terdefine, make lakukan nomor 3, dan apabila tidak make gunakan model standard
3. jadi A itu sendiri ada , nya. make lakukan

contoh:
```#define LYM_SETLOG {}, {}

#ifndef LYM_SETLOG
#define LYM_SETLOG {default}, default
#endif

#define ___LYM_VALOG_(x, y) typedef enum {x} LYE_LYLOG_MSG; typedef enum {y} LYE_LYLOG_COLOR;
#define ___LYM_VALOG(...) ___LYM_VALOG_(__VA_ARGS__)

___LYM_VALOG(LYM_SETLOG)```
